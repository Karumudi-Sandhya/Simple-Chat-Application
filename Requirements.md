## Requirements – Simple Chat Application

## Software Requirements

* Operating System: Windows, Linux, or macOS

* Web Browser: Google Chrome, Microsoft Edge, Firefox, or Safari

* Code Editor: Visual Studio Code or any text editor

* Web Technologies: HTML, CSS, JavaScript

* Internet Connection: Not required to run the app

## Hardware Requirements

* Processor: Intel / AMD or equivalent

* RAM: Minimum 4 GB

* Storage: At least 100 MB free space

* Display: Minimum resolution of 1366 × 768

* Input Devices: Keyboard and mouse or touchpad
  
## Technologies Used

* HTML – Structure of the chat interface

* CSS – Styling and layout design

* JavaScript – Chat logic, DOM manipulation, and interactivity

* Browser Local Storage – Saving chat messages

## Functional Requirements

* The system shall allow users to enter a username
  
* The system shall allow users to send chat messages

* The system shall display messages with timestamps

* The system shall allow users to edit, Delete messages.

* The system shall store chat messages using local storage

* The system shall update the chat window dynamically

## Non-Functional Requirements

* The application shall be user-friendly

* The system shall respond instantly to user input

* The application shall be lightweight and fast

* The system shall be compatible with modern browsers

* The application shall ensure message persistence

* The system shall not require backend services
